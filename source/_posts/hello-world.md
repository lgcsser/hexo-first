## hexo前置环境
* nodejs和npm包管理工具
* git

## hexo前置技能
- github账号
- ssh配置
- 开源项目github托管
- Markdown编辑语法类似wiki

## hexo命令工具
```
1.hexo innit 工程名称
2.hexo server 开启本地服务
3.hexo generate 静态化文件
4.hexo deploy 部署命令
5.hexo new 文章名称
```
